# 数据库建模

## 维度建模
+ 维度
  - 缓慢变化维（SCD）：维度数据随时间推移而缓慢变化，eg:用户信息、商品信息
  - 退化维：将维度表中的某些字段直接放到事实表中，eg:日期表中的星期几、月份、季度、年份等
  - 层次维：将维度表中的某些字段进行分层，eg:地区维度表中的国家、省份、城市、区县等
  - 父子维：将维度表中的某些字段进行父子关系，eg:部门维度表中的上级部门、下级部门等
+ 模型
  - 星型模型：事实表和维度表之间是一对多的关系，eg:用户购买商品
  - 雪花模型：事实表和维度表之间是多对多的关系，eg:用户购买商品
## 数据库分层理论
+ ODS:操作数据存储，存储原始数据，数据清洗、数据转换、数据加载
+ DWD:数据仓库明细层，存储清洗、转换后的数据
+ DWS:数据仓库汇总层，存储汇总后的数据
+ ADS:数据仓库应用层，存储最终的数据，供业务使用
