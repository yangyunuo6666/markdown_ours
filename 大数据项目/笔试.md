# 笔试

## 数据结构与算法
+ 栈先进后出，先进先出，队列(常用链表实现)先进先出
+ 排序算法的时间复杂度
  - 快速排序O(nlogn)，最好：O(nlogn)，最坏：O(n^2)
  - 冒泡排序O(n^2)，最好：O(n)，最坏：O(n^2)
  - 插入排序O(n^2)，最好：O(n)，最坏：O(n^2)
  - 希尔排序O(n^1.3)，最好：O(n)，最坏：O(n^2)
  - 桶排序O(n)，最好：O(n)，最坏：O(n^2)
  - 归并排序O(nlogn)
  - 堆排序O(nlogn)
  - 选择排序O(n^2)
  - 基数排序O(nk)
  - 计数排序O(n+k)
+ 二叉树
  - 在N层节点数为2^N-1，满二叉树
+ DAG
  - 最小生成树MST边数：n-1
## java基础
+ == 与 string.equals()的区别
  - ==比较的是两个对象的内存地址，string.equals()比较的是两个字符串的内容 
+ java中两种创建string对象的方式
  - String str = "hello"; // 若常量池中没有"hello"，则创建一个"hello"对象并引用，否则直接引用
  - String str = new String("hello"); // 在堆内存中创建一个新的String对象并引用
## python基础
+ python内存机制：对象共享内存，变量保存的仅仅对象在内存中的地址
  - a = [1, 2, 3]  b = a # a和b指向同一个对象(因他们保存的是同一内存地址)
+ 可变对象机制：a：可保存任何类型的对象。
